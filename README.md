# 📢 Campus Noticeboard

This is a simple noticeboard web app I built where anyone can view notices, and only authorized users can add new ones using a password.

---

## 🚀 Live Demo

You can check the live project here:
👉 https://campusnotice.netlify.app/

---

## ✨ What it can do

* Add new notices (password required)
* Delete existing notices
* Clean and simple interface
* Stores data in browser using localStorage (no backend)

## 📸 How to use it

1. Click on **"Unlock Add Notice"**
2. Enter the password
3. Add your notice
4. It will instantly show up on the board

---

## ⚠️ Note

This project uses localStorage, so all data is saved only in your browser.

Also, the password is written in the frontend code, so this is just for learning/demo purposes — not suitable for real-world secure use.
